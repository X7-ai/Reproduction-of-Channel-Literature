# Reproduction-of-Channel-Literature

包含三个主要的内容：文献内容的精炼化整理，关于定位网络的代码，本人在阅读中产生的疑问以及解答；
文献来源：IEEE TRANSACTIONS ON WIRELESS COMMUNICATIONS, VOL. 23, NO. 7, JULY 2024
