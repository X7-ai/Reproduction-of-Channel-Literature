# Reproduction-of-Channel-Literature
1.这里主要记录整理了对我在大二大三时期阅读文献《From Data-Driven Learning to Physics-Inspired Inferring: A Novel Mobile MIMO Channel Prediction Scheme Based on Neural ODE》时做的笔记和复现的内容。

2.包含三个主要的内容：文献内容的精炼化整理，关于定位网络的代码，本人在阅读中产生的疑问以及解答；

3.文献来源：IEEE TRANSACTIONS ON WIRELESS COMMUNICATIONS, VOL. 23, NO. 7, JULY 2024；

4.特殊说明，由于本人电脑算力有限，无GPU，同时原文献并未公开其代码和数据，在复现中，考虑到核心目的是为了学习lstm网络，故我设计了一个小的数据，以此对比lstm和cnn在预测这种时间序列的差异。
